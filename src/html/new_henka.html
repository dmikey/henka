HENKA

responsive all the things! (resize this page to watch it respond)

Light weight, ultra portable, responsive javascript library — henka will detect view-port size on initial load of the page, and when the window is resized.

Small compact, and single purpose - add classes to elements based on a breakpoint. Nothing more.

Compatible IE 6, 7, 8, 9, 10. Safari, Chrome, Firefox.

Get started fast, no additional Javascript to write.

Add a script tag to the source 

     <script src="henka.js">
     </script>

Markup the body tag with breakpoints. 

    <body
    data-henka='
        [300, 600, 900, 1200]
    '/>

Event driven callbacks. 

    <script>
     var _henka = henka.noConflict();
     _henka.onUpdate(
     function(breakpoint){
          //breakpoint passed

     });
